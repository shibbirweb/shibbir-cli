# Commands

The CLI is menu-driven. Run `shibbir` and navigate the prompts.

## Menu tree

```
shibbir
├── Node
│   └── Package Development
│       └── Bump Version
└── Windows
    └── Network
        └── WinNAT
            ├── Stop
            └── Start
```

Every submenu has a **Back** option to return to the previous menu; the top menu has **Exit** to quit. Pick the wrong menu? Choose **Back**.

## Index

| Command | Category | Description |
