# Installation

## Requirements

- Node.js 18 or newer

## Install globally

```bash
npm install -g shibbir
```

Then run:

```bash
shibbir
```

## Run without installing

```bash
npx shibbir
```

## Run from source (development)

```bash
git clone https://github.com/shibbirweb/shibbir-cli.git
cd shibbir-cli
npm install
npm start        # runs src/index.ts via ts-node
```

See **[[Adding a Command]]** to extend the CLI.
